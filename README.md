# automated-ai-scientist
An automated ai scientist developed during the Gradient Descent Hackathon, Stockholm, 27-28 sep 2024



# Run

```bash 
python3 -m venv .venv
```


```bash 
source .venv/bin/activate
```


```bash 
pip3 install -r requirements.txt 
```
